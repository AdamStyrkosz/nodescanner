Wszystko czego potrzebujesz

Proste narzędzia do zarządzania projektami Node.js w jednym miejscu.

Skanowanie dysku

Automatycznie znajdź wszystkie projekty Node.js na dysku. Skanuj dowolny folder i odkryj zapomniane projekty.

Jednym kliknięciem

Uruchamiaj i zatrzymuj projekty bez otwierania terminala. Obsługuje npm run dev i npm start automatycznie.

Wbudowany terminal

Obserwuj logi w czasie rzeczywistym bezpośrednio w aplikacji. Bez przełączania między oknami.

Monitor portów

Zobacz które porty są zajęte przez procesy Node.js i zamknij je jednym kliknięciem.

Cross-platform

Działa na Windows, macOS i Linux. Zbudowane z Electron dla natywnego doświadczenia.

Open Source

Kod źródłowy dostępny na GitHub. Darmowy na zawsze, bez ukrytych kosztów.

Jak to działa?

Trzy proste kroki do efektywnego zarządzania projektami.

1

Wybierz folder

Wskaż folder który chcesz przeskanować — może to być cały dysk lub konkretny katalog z projektami.

2

Przeglądaj projekty

NodeScanner automatycznie znajdzie wszystkie projekty z package.json i wyświetli je na liście.

3

Uruchom i pracuj

Kliknij Start przy wybranym projekcie i obserwuj logi w wbudowanym terminalu.
